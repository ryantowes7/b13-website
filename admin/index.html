Konfigurasi Backend

Pastikan konfigurasi config.yml Anda sudah benar:

# config.yml
backend:
  name: git-gateway
  branch: main
  repo: your-username/your-repo-name # Ganti dengan repo Anda

# Pengaturan Identity
local_backend: true # Untuk development lokal

# Izinkan semua pengguna yang diundang
publish_mode: editorial_workflow
