Langkah Cepat Perbaikan

1. Periksa Git Gateway di Netlify

Pastikan Git Gateway sudah diaktifkan di: Netlify Dashboard → Site Settings → Identity → Services → Git Gateway

2. Periksa Repository Connection

Pastikan repository GitHub terhubung dengan benar di: Netlify Dashboard → Site Settings → Deploy → Repository connection

3. Periksa File config.yml

Pastikan file config.yml ada di folder /admin/ dan berisi konfigurasi yang benar:

backend:
  name: git-gateway
  branch: main
media_folder: "assets/uploads"
public_folder: "/assets/uploads"

4. Clear Cache & Cookies

Coba buka halaman admin di mode incognito/private browsing untuk mengecualikan masalah cache.
