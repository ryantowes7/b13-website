  • Masalah dengan konfigurasi CMS
  • Blokir oleh kebijakan keamanan browser

Langkah-langkah Perbaikan:

  1. Refresh halaman ini (Ctrl+F5 untuk refresh lengkap)
  2. Pastikan Anda menggunakan browser modern (Chrome, Firefox, Edge)
  3. Coba buka di mode incognito/private browsing
  4. Nonaktifkan ekstensi browser yang mungkin memblokir script
  5. Hubungi administrator jika masalah berlanjut

Konfigurasi yang Diperlukan:

Pastikan Anda telah mengaktifkan Git Gateway di Netlify:

  1. Buka Settings → Identity → Services → Git Gateway
  2. Klik Enable Git Gateway
  3. Pilih provider yang sesuai (biasanya GitHub)
  4. Autorisasi akses ke repository Anda

Pastikan file config.yml Anda berisi:

backend:
  name: git-gateway
  branch: main

publish_mode: editorial_workflow
media_folder: "assets/uploads"
public_folder: "/assets/uploads"
